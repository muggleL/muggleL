curl 常见用法

DG / 2019-12-07

在 Linux/unix 中，curl 是最常用到的 web 调试工具。Bite Size Networking 有一张漫画详细地解释了 curl 的参数。

请求一个地址

curl https://httpbin.org/anyting

{
  "args": {},
  "data": "",
  "files": {},
  "form": {},
  "headers": {
    "Accept": "*/*",
    "Host": "httpbin.org",
    "User-Agent": "curl/7.64.1"
  },
  "json": null,
  "method": "GET",
  "origin": "11*.*9.3*.72, 11*.*9.3*.72",
  "url": "https://httpbin.org/anything"
}

可见 curl 的默认 User-Agent 为 curl 加版本号，可以通过识别此标记特定地针对 curl 返回适合终端查看的内容比如 ip.sb

发送 GET 请求

curl -X GET https://httpbin.org/anything

{
  "args": {},
  "data": "",
  "files": {},
  "form": {},
  "headers": {
    "Accept": "*/*",
    "Host": "httpbin.org",
    "User-Agent": "curl/7.64.1"
  },
  "json": null,
  "method": "GET",
  "origin": "11*.*9.3*.72, 11*.*9.3*.72",
  "url": "https://httpbin.org/anything"
}

通过 GET 发送参数

curl -X GET -d 'value=panda' https://httpbin.org/anything

{
  "args": {},
  "data": "",
  "files": {},
  "form": {
    "value": "panda"
  },
  "headers": {
    "Accept": "*/*",
    "Content-Length": "11",
    "Content-Type": "application/x-www-form-urlencoded",
    "Host": "httpbin.org",
    "User-Agent": "curl/7.64.1"
  },
  "json": null,
  "method": "GET",
  "origin": "11*.*9.3*.72, 11*.*9.3*.72",
  "url": "https://httpbin.org/anything"
}

value 作为 form 表单被目标网站接受到了。

修改请求头

curl -X GET -H 'User-Agent:elephant' https://httpbin.org/anything

{
  "args": {},
  "data": "",
  "files": {},
  "form": {},
  "headers": {
    "Accept": "*/*",
    "Host": "httpbin.org",
    "User-Agent": "elephant"
  },
  "json": null,
  "method": "GET",
  "origin": "11*.*9.3*.72, 11*.*9.3*.72",
  "url": "https://httpbin.org/anything"
}

参数中间用冒号。

获取响应头

curl -I https://httpbin.org/anything

大写的 i

Get 发送 json 数据

curl -X GET -H "Content-type: application/json" -d '{"value":"panda"}' https://httpbin.org/anything

{
  "args": {},
  "data": "{\"value\":\"panda\"}",
  "files": {},
  "form": {},
  "headers": {
    "Accept": "*/*",
    "Content-Length": "17",
    "Content-Type": "application/json",
    "Host": "httpbin.org",
    "User-Agent": "curl/7.64.1"
  },
  "json": {
    "value": "panda"
  },
  "method": "GET",
  "origin": "11*.*9.3*.72, 11*.*9.3*.72",
  "url": "https://httpbin.org/anything"
}

一般没人用 GET 发送 json。

Accept-Encoding

curl -X GET -H 'Accept-Encoding:gzip'  https://httpbin.org/anything --output test.gzip

会将 请求得到的结果以 gzip 压缩格式保存成 test.gzip 文件

发送文件中的 json 

echo '{"value":"panda"}' > test.json
ls test.json

{"value":"panda"}

curl -X GET -H 'Content-Type:application/json' -d @test.json  https://httpbin.org/anything

{
  "args": {},
  "data": "{\"value\":\"panda\"}",
  "files": {},
  "form": {},
  "headers": {
    "Accept": "*/*",
    "Content-Length": "17",
    "Content-Type": "application/json",
    "Host": "httpbin.org",
    "User-Agent": "curl/7.64.1"
  },
  "json": {
    "value": "panda"
  },
  "method": "GET",
  "origin": "11*.*9.3*.72, 11*.*9.3*.72",
  "url": "https://httpbin.org/anything"
}

请求图片

curl -X GET -H 'Accept:image/png'  https://httpbin.org/image --output test.png

图片保存为 test.png
